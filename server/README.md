# Poker Online — Hold'em Club

Браузерный Texas Hold'em: профиль, рейтинг, монеты, очередь на 4 игрока и приватные комнаты.

## Возможности

- **ID + ник** — стабильный `playerId` в `localStorage` (из Unity: `?pid=&queue=1`)
- **Монеты** — старт **50 000**, взнос в онлайн-матч **1 000**, победитель получает **4 000**
- **Очередь** — «Играть онлайн» → поиск; при **4** игроках автоматический стол
- **Рейтинг (MMR)** — стартовый 1000, Elo после конца матча
- **Приватная комната** — ссылка `/?room=XXXXXX`, хост стартует вручную

## Запуск локально

```bash
cd server
npm install
npm start
```

Открой http://localhost:8787

API: `/health`, `/api/online`, `/api/leaderboard`

Данные профилей: `server/data/profiles.json`

## Unity

Меню «Играть онлайн» открывает браузер с вашим ID и автопоиском матча (`queue=1`).
