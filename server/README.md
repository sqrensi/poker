# Poker Online — браузерный матч по ссылке

## Как это работает

1. Хост открывает сайт, жмёт **«Создать стол»**
2. Получает **ссылку** `http://…/?room=XXXXXX` → кидает друзьям
3. Друзья открывают ссылку в браузере и попадают за тот же стол
4. Хост жмёт **«Начать матч»** — все играют в браузере

Unity здесь только для режима «с ботами». Онлайн — целиком веб.

## Запуск

```bash
cd server
npm install
npm start
```

Открой http://localhost:8787

## Друзья не в одной Wi‑Fi

Нужен публичный адрес (ngrok / VPS) и:

```bash
PUBLIC_BASE=https://ваш-адрес npm start
```

Тогда в ссылке будет этот адрес, а не localhost.
